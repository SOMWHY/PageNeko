## 关于贡献

谁可以贡献？

任何想为这个可爱的小猫项目做出改进的人——无论是初学者还是有经验的开发者——都可以参与贡献。

如何贡献？

1. 先找一个 issue（或者新建一个 issue）来修复或实现功能。
2. 将本仓库 Fork 到你的 GitHub 账号下。
3. 将 Fork 克隆到本地机器。
4. 进行修改或改进。
5. 添加、提交并推送你的更改：
   ```bash
   git add .
   git commit -m "描述你的修改"
   git push origin your-branch-name
   ```
6. 提交一个 Pull Request，等待合并即可。

注意事项

1. 请勿在提交中更改与本次改动无关的内容。
2. 如果你更新了注释或使用说明，请同步更新 README 中的“如何自定义”章节。
3. 在开始编码之前，如有疑问欢迎随时提问。
